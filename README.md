# PosterGenerator
画像を減色してSchematicファイルに変換する  

## Requires  
使用モジュール  
 - [numpy](https://github.com/numpy/numpy)
```bash
$ pip install numpy
```
 - [nbtlib](https://github.com/vberlier/nbtlib)  
```bash
$ pip install nbtlib
```
 - opencv-python
```bash
$ pip install opencv-python
```

## Usage  
```bash
$ python PosterGenerator.py example.png
```
